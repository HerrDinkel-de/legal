# Legal Pages - HerrDinkel.de

Dieses Repository enthält die rechtlich erforderlichen Seiten (**Impressum** und **Datenschutzerklärung**) für das Netzwerk **HerrDinkel.de** (Minecraft-Server, Discord, Bewerbungsportal & Webdienste).

Die Seiten werden automatisch über **Cloudflare Workers / Pages** unter der Subdomain `legal.herrdinkel.de` bereitgestellt und abgesichert.

## 🛠️ Technische Infrastruktur

Die Datenschutzerklärung deckt folgende genutzte Dienste & Partner ab:

* **CDN & DDoS-Schutz:** Cloudflare Inc.
* **Root- Server / Gameserver:** OVH SAS (Frankreich)
* **Mail-Server:** MC-Host24 GmbH (Deutschland)
* **Bewerbungsportal:** Team-Panel
* **Community-Plattformen:** Discord Inc. / Minecraft-Server Log-Systeme

## 🔄 Aktualisierung

Änderungen an den HTML-Dateien in diesem Repository werden bei jedem `git push` auf den `main`-Branch automatisch durch Cloudflare neu deployed und sind innerhalb weniger Sekunden live.
